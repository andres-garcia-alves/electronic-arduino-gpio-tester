# Arduino GPIO Tester

Electronic proyect for fast test of Arduino analog and digital (GPIO) pins.  

&nbsp;

This project use the following electronic components:
- 1 x Arduino Nano v3.0
- 10 x LED 5mm
- 10 x 2k ohm resistors
