# Arduino GPIO Tester

Electronic proyect for easy test of Arduino Nano/UNO analog and digital (GPIO) pins.  

&nbsp;

This project use the following electronic components:
- 1 x Arduino Nano v3.0
- 2 x Connector pins
- 22 x LED 3mm
- 22 x 1k ohm resistors
